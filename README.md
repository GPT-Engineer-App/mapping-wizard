# mapping-wizard

a tool to upload spreadsheet or csv file that has a mapping template, column A the new column column B is the prompt matching instructions and column C will be the data template headers. Then upload the 2nd file to match it and generate the new template

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/mapping-wizard.git
cd mapping-wizard
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
